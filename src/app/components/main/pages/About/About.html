IWrappers is the library for creating a modern UI, similar to a desktop OS. You can create stretchable, draggable, dropped elements of interface. For example, it can be windows, chats, panels, and other.
All components have flexible settings that can be customized.

The following components are available:

DnDWrapper is the component for drag and drop logic, as well as drag files.

MovableWrapper - the component for moving around the screen or a specified area. It can be useful for creating a modal window or chat.

ResizableWrapper is the component for resizing a block, it can be useful for creating panels.
